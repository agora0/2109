“我们在新闻报道中总是出现一些不好的事情，但现在我们也会报道一些不错的消息，”来自拉马拉的22岁的学生萨亚说 [ANNE PAQ / 半岛电视台]

巴勒斯坦，伯利恒–数万人参加了巴勒斯坦的马拉松赛，这是在约旦河西岸伯利恒地区举行的年度活动。

与往年一样，3月23日举行的马拉松比赛（该赛事的第六版）吸引了来自西岸和耶路撒冷以及海外各地的人们。

对许多外国人来说，这次活动是首次能来到巴勒斯坦，目睹巴勒斯坦人生活条件的机会。

来自英格兰的29岁的Deena说：”今天这种感觉真是太棒了，能来到这里并与巴勒斯坦人站在一起。”

“我们沿着城墙散步，看到隔离的土地。我们看到了Aida难民营及其贫困，但我们也感到巴勒斯坦人是多么善良，这是我第一次来到巴勒斯坦，我感到羞愧，因为我曾经对它了解得太少，这很重要，表示声援，”Deena补充说。

据该活动网站称，其目标是”将在巴勒斯坦自由行动的基本权利置于镜头下”。

据联合国人道主义事务协调厅（OCHA，United Nations Office for theCoordination of Humanitarian Affair）称，约旦河西岸的道路上设置了500多个障碍物–检查站、路障和土墩等，用以限制巴勒斯坦车辆的行驶。

这些也限制了马拉松参与者的移动。为了完成42公里的比赛，参赛者必须完成相同路线的两个回路，以避免遇到以色列的军事检查站或定居点。

有些人甚至利用上述情况来强调巴勒斯坦人人权被侵犯，以此声援巴勒斯坦人。

马拉松赛也是促进巴勒斯坦旅游业的一种方式，本次马拉松还举办了许多其他活动。

约有7千注册参加今年的马拉松比赛，乌克兰的尤里·弗拉季吉耶（Yuri Vladigier）获胜。


巴勒斯坦马拉松赛吸引了7千名参赛者。 在短短的六年时间里，该项比赛获得了国际认可。 然而，许多巴勒斯坦人不能参加这次活动，因为加沙仍然处于围困之中。 每年能得到的许可证很少，而许多巴勒斯坦难民由于以色列的限制而无法参与 [ANNE PAQ / 半岛电视台]

马拉松为伯利恒带来了欢庆和享受，许多团体和志愿者参与 [ANNE PAQ / 半岛电视台]

还有一场5公里的比赛，每个人都可以参加，包括家人和孩子。 全程马拉松选手必须完成相同路线的两圈，因为在巴勒斯坦权力机构控制下, 组织者无法找到连续42千米的单程赛程 [ANNE PAQ / 半岛电视台]

“这是我第一次来到巴勒斯坦，我在这里只待了四天，”来自苏格兰的26岁的彼得说。 “我们去了纳布卢斯，希伯伦，耶路撒冷的各个地方，我们被希伯伦的定居者追赶，能实际上看到一个人因为种族而恨另一个人，这实在令人震惊。” [ANNE PAQ / 半岛电视台]

在西岸，巴勒斯坦人的行动自由受到严重限制。 隔离墙和西岸的众多军事检查站对巴勒斯坦人的生活具有破坏性影响。[ANNE PAQ / 半岛电视台]

“今天我很兴奋，你看到每个人都在一起，各种各样的人。我们想表明我们在这里，我们应该有权利活动。今天，对我们和残疾人来说，都是额外美好的，街道是开放的。今天你感觉你可以自由了，”坐在轮椅上的谢林来自伯利恒。[ANNE PAQ / 半岛电视台]

一名参与者举起支持被监禁少年Ahed Tamimi的标志。 后者是在Nabi Saleh村庄的一位年轻的活跃分子，她在2017年12月19日晚，被以色列当局逮捕，因此前一个广为流传的视频中，她与家门口的她两名以色列士兵爆发了冲突。 她被判处八个月监禁。[ANNE PAQ / 半岛电视台]


“今天我们向全世界传递的信息是，耶路撒冷永远是巴勒斯坦的首都，耶路撒冷不会被巴勒斯坦人民抛弃，行动自由是巴勒斯坦人的权利，我们将实现这一权利，这种占领不会使我们 害怕，”来自Al-Azzeh营的50岁的Mazen说 [ANNE PAQ / 半岛电视台]

马拉松路线经过Aida难民营，在那里，一些妇女将狱中亲人的照片拿在手里，静静坐着。以色列部队会定期在夜间进入营地逮捕居民。[ANNE PAQ / 半岛电视台]

Aida的大钥匙象征着巴勒斯坦难民的返回权。[ANNE PAQ / 半岛电视台]

以色列的隔离墙距离Aida营地住宅仅几米远。[ANNE PAQ / 半岛电视台]

“我们是生活在占领下的人，这里一切都是禁止的，我们对世界说，我们是自由的人，我们有权像所有人一样生活，”45岁的萨米在伯利恒说。[ANNE PAQ / 半岛电视台]


行动自由是一项普遍的人权。 《世界人权宣言》第13条规定，”每个人都有在各国境内自由行动和居住的权利”。[ANNE PAQ / 半岛电视台]
